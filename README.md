# Simplifying Compliance for Research Analysts

A pixel-perfect, highly responsive landing page designed exactly according to Figma mockups and specifications. This page is built using modern front-end web standards (semantic HTML5, clean vanilla CSS3, and JavaScript) and is powered by Vite for an optimized development workflow.

## 🚀 Live Demo & Repository
- **GitHub Repository**: [https://github.com/vaibhav-pruthi/-Figma-Compliance-](https://github.com/vaibhav-pruthi/-Figma-Compliance-)

---

## 🛠️ Tech Stack
- **Core Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom Design Tokens, Flexbox, CSS Grid)
- **Logic**: Vanilla JavaScript
- **Dev Server / Bundler**: Vite (for rapid hot-reloading and asset compilation)

---

## 📐 Responsive Breakpoints & Figma Accuracy

This project has been meticulously optimized for the following breakpoints to ensure no text overlapping, correct image scaling, and a fully functional layout at all sizes:
- 🖥️ **Desktop (1400px / 1280px)**: Two-column grid layout, visible desktop navigation, full-sized logo, and placed visual decorations.
- 📁 **Tablet (768px)**: Optimized tablet grid structure, collapsed hamburger navigation menu, and scaled-down decorative elements.
- 📱 **Mobile (375px)**: Single-column layout, full screen mobile menu overlay with vector navigation icons, and properly scaled typography for readable mobile viewports.

## ⚙️ Local Development

To run the project on your local machine, follow these simple steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed (v18.0.0 or higher recommended).

### Steps
1. **Clone the repository**:
   ```bash
   git clone https://github.com/vaibhav-pruthi/-Figma-Compliance-.git
   cd -Figma-Compliance-
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the local development server**:
   ```bash
   npm run dev
   ```

4. **Open the browser**:
   Navigate to `http://localhost:5173/` in your browser.
